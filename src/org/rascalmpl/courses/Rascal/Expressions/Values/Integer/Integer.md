#### Syntax

#### Description

The integer values are represented by the type `int` and are written as usual. They can be _arbitrarily large_.

See ((Values-Number)) for all operations and functions on integers, reals and numbers.

#### Examples

*  `12`
*  `0`
*  `-123456789`
