#### Synopsis

Less than operator on string values.

#### Syntax

`Exp~1~ < Exp~2~`

#### Types


| `Exp~1~` | `Exp~2~` | `Exp~1~ < Exp~2~`  |
| --- | --- | --- |
| `str`     |  `str`    | `bool`                |


#### Function

#### Description

Yields `true` if the string value of _Exp_~1~ is strictly lexicographically less
than the string value of _Exp_~2~, and `false` otherwise.

#### Examples

```rascal-shell
"abc" < "abcdef";
"abc" < "defghi";
"abc" < "a";
```
