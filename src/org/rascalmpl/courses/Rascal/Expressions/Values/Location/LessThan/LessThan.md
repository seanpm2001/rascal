#### Synopsis

The less than operator on location values.

#### Syntax

`Exp~1~ < Exp~2~`

#### Types


| `Exp~1~` | `Exp~2~` | `Exp~1~ < Exp~2~`  |
| --- | --- | --- |
| `loc`     |  `loc`    | `bool`                |


#### Function

#### Description

Yields `true` if the location value of _Exp_~1~ is strictly textually contained
in the location value of _Exp_~2~, and `false` otherwise.
