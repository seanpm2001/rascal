#### Synopsis

The _not_ operator on Boolean values.

#### Syntax

`! Exp`

#### Types

//

| `Exp` | `! Exp`  |
| --- | --- |
| `bool`   | `bool`     |


#### Function

#### Description

The _not_ operator on Boolean values defined as follows:

| `Exp`  | `! Exp`  |
| --- | --- |
| `true`   | `false`    |
| `false`  | `true`     |


#### Examples

```rascal-shell
!true;
```
