#### Synopsis

Addition on numeric values.

#### Syntax

`Exp~1~ + Exp~2~`

#### Types


| `Exp~1~`  |  `Exp~2~` | `Exp~1~ + Exp~2~`  |
| --- | --- | --- |
| `int`      |  `int`     | `int`                 |
| `int`      |  `real`    | `real`                |
| `real`     |  `real`    | `real`                |


#### Function

#### Description

Yields the numerical sum of the values of _Exp_~1~ and _Exp_~2~.

#### Examples

```rascal-shell
12 + 13
12 + 13.5
```
