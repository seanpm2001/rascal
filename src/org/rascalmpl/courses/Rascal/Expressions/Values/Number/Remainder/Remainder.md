#### Synopsis

Remainder of two integer values.

#### Syntax

`Exp~1~ % Exp~2~`

#### Types


| `Exp~1~`  |  `Exp~2~` | `Exp~1~ % Exp~2~`  |
| --- | --- | --- |
| `int`      |  `int`     | `int`                |


#### Function

#### Description

Yields the remainder when dividing the of _Exp_~1~ by the value of _Exp_~2~.

#### Examples

```rascal-shell
12 % 5
12 % 6
```

#### Benefits

#### Pitfalls

Remainder is only defined on integers:
```rascal-shell,error
13.5 % 6
```
