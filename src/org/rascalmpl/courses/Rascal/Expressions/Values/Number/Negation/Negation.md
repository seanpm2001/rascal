#### Synopsis

Negate a numeric value.

#### Syntax

`- Exp`

#### Types


| `Exp`  |  `- Exp`  |
| --- | --- |
| `int`    |  `int`      |
| `real`   |  `real`     |


#### Function

#### Description

Yields the negated values of _Exp_.

#### Examples

```rascal-shell
-12
-13.5
- -12
```
